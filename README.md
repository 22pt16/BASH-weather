# BASH-weather
weather forecasting application using bash 
This makes use of <b>API request<b> using wget cmd
that calls a HTTP request with OpensourceAPI Realtime weather APIs to fetch details of current weather at a given location.

<p>The Location can either :
<ul>
  <li>Entered Manually</li>
  <li>Found through IP</li>
</ul></p>

<p>APIs used</p>
<ul>
  <li>IP finder for present active device</li>
  <li>Location's Latitude and Longtitude finding by IP</li>
  <li>Fetch Realtime Weather of location by opensrc api</li>
</ul>

<p>
  The weather api fetches JSON and html file of details of weather forecast which is parsed and displayed in terminal with colorful appealing elements.
  Forecast of upto 2 days can be predicted.
</p>
